# fitness-check 專案說明

> 最後更新：2026-06-17

---

## 1. 用途與目標

**獨立靜態網頁工具**，與評鑑模組（pe-tools）完全分開部署。

主要功能：
- 老師將各班體適能成績 xlsx/xlsm 拖曳上傳，自動對照教育部百分位常模
- 顯示每位學生 4 個項目（坐姿體前彎、仰臥起坐、立定跳遠、800公尺跑走）的百分位等級（PR）
- 標示是否及格（四項均 ≥ 25th 百分位）
- 支援調整未及格學生成績、下載調整後 xlsx

**及格標準：** 四項均達 25th 百分位以上

**適用年級：** 4～6 年級（年齡對應 10～12 歲）

**部署網址：** https://asics49.github.io/fitness-check/

**GitHub Repo：** https://github.com/asics49/fitness-check

## 2. 已完成功能

### 核心功能
- [x] 拖曳或點擊上傳 xlsx/xlsm 檔案（使用 SheetJS 瀏覽器端解析）
- [x] 自動偵測欄位名稱（支援不同工作表格式，不靠固定欄位位置）
- [x] 計算學生年齡（民國年7碼出生日期 → 西元 → 年齡）；若無出生年月日欄位則依年級推算（4年級=10歲、5年級=11歲、6年級=12歲）
- [x] 800公尺跑走格式轉換：m.ss（3.17 = 3分17秒）→ 秒數比對常模 → 分秒顯示

### 常模資料（內嵌於 index.html）
- [x] 坐姿體前彎：教育部常模（男女 × 10/11/12 歲 × 19 個百分位）
- [x] 仰臥起坐 60 秒：從教育部體適能網站 PDF 取得（原始資料無此常模，新增）
- [x] 立定跳遠：教育部常模
- [x] 800公尺跑走：教育部常模（lower = better，邏輯與其他三項相反）

### 介面功能
- [x] 及格率統計列（全校＋四/五/六年級，含進度條與顏色）
- [x] 年級 tab 切換（全部 / 四年級 / 五年級 / 六年級）
- [x] 班級、性別篩選
- [x] 顯示篩選（全部 / 未及格 / 四項均<25th / 全部及格 / 優良≥75th）
- [x] 每位學生顯示「及格」/「未及格」標籤
- [x] PR 顏色編碼：紅（<25th）/ 橙（25–50th）/ 黃（50–75th）/ 綠（≥75th）
- [x] 統計卡：顯示人數、已填成績、未及格人數、及格人數、及格率

### 調整模式
- [x] 點擊「調整未及格成績」按鈕 → 自動篩出未及格學生
- [x] 分數欄變成橘色 input，可直接修改數值
- [x] PR 值與及格標籤即時更新
- [x] 已調整的列顯示黃色外框
- [x] 調整人數計數顯示

### 下載功能
- [x] 下載調整後 xlsx（使用 SheetJS 寫出）
- [x] 可選全校或單一年級
- [x] 輸出欄位與原始上傳格式相符：班級、座號、學號、姓名、性別、出生年月日、坐姿體前彎、立定跳遠、仰臥起坐、800公尺跑走
- [x] 800公尺轉回 m.ss 格式輸出（e.g., 197秒 → 3.17）
