# fitness-check 專案說明

> 最後更新：2026-06-17

---

## 1. 用途與目標

**獨立靜態網頁工具**，與評鑑模組（pe-tools）完全分開部署。

主要功能：
- 老師將各班體適能成績 xlsx/xlsm 拖曳上傳，自動對照教育部百分位常模
- 顯示每位學生 5 個項目（坐姿體前彎、仰臥起坐、仰臥捲腹、立定跳遠、800公尺跑走）的百分位等級（PR）
- 標示是否及格（四項均 ≥ 25th 百分位）
- 支援調整未及格學生成績、自動調整及格率、下載調整後 xlsx
- 匯出教育部體適能上傳格式（4–6 年級）及身高體重上傳格式（1–6 年級）

**及格標準：** 四項均達 25th 百分位以上

**適用年級：** 體適能對照 4～6 年級（10～12 歲）；身高體重匯出 1～6 年級

**部署網址：** https://asics49.github.io/fitness-check/

**GitHub Repo：** https://github.com/asics49/fitness-check

**存取密碼：** 頁面有密碼保護，密碼存放於 index.html `pwScreen` 區塊（勿公開）

## 2. 已完成功能

### 核心功能
- [x] 拖曳或點擊上傳 xlsx/xlsm 檔案（使用 SheetJS 瀏覽器端解析）
- [x] 自動偵測欄位名稱（支援不同工作表格式，不靠固定欄位位置）
- [x] 計算學生年齡（民國年7碼出生日期 → 西元 → 年齡）；若無出生年月日欄位則依年級推算（4年級=10歲、5年級=11歲、6年級=12歲）
- [x] 800公尺跑走格式轉換：m.ss（3.17 = 3分17秒）→ 秒數比對常模 → 分秒顯示

### 常模資料（內嵌於 index.html，`const NORM = {...}`）
- [x] 坐姿體前彎（bend）：教育部常模，男女 × 10/11/12 歲 × 19 個百分位
- [x] 仰臥起坐（situp）：來源 fitness.org.tw muscular-fit 頁面（2026-06-17 更新）
- [x] 仰臥捲腹（curlup）：來源 fitness.org.tw curl-up 頁面（2026-06-17 新增）
- [x] 立定跳遠（jump）：教育部常模
- [x] 800公尺跑走（run）：教育部常模（lower = better，邏輯與其他項目相反）

> **注意：** 仰臥起坐（situp）已於 2026-06-17 以 fitness.org.tw 官方數據重新校正，舊版數值有誤。

### 介面功能
- [x] 及格率統計列（全校＋四/五/六年級，含進度條與顏色）
- [x] 年級 tab 切換（全部 / 四年級 / 五年級 / 六年級）
- [x] 班級、性別篩選
- [x] 顯示篩選（全部 / 未及格 / 四項均<25th / 全部及格 / 優良≥75th）
- [x] 每位學生顯示「及格」/「未及格」標籤
- [x] PR 顏色編碼：紅（<25th）/ 橙（25–50th）/ 黃（50–75th）/ 綠（≥75th）
- [x] 統計卡：顯示人數、已填成績、未及格人數、及格人數、及格率

### 調整模式
- [x] 手動調整：點擊「調整未及格成績」→ 分數欄變橘色 input，PR 與及格標籤即時更新
- [x] 自動調整及格率：輸入各年級目標 %，系統自動選出最接近及格的學生調整至 25th 門檻
  - 排序邏輯：失敗項目最少優先 → PR 總分最高優先
  - 調整值：NORM 陣列 index 4（= 最低及格門檻，getPR 回傳 25th）
- [x] 已調整的列顯示黃色外框（手動與自動調整共用標示機制）

### 匯入功能
- [x] **學生基本資料**（含身分證字號）：主畫面「📋 匯入基本資料」按鈕，`BASIC_DATA` 物件以學號為 key
  - 自動偵測身分證誤填到學號欄（台灣身分證格式 `/^[A-Za-z][0-9]{9}$/`）並對調修正
- [x] **身高體重資料**：主畫面「⚖️ 匯入身高體重」按鈕，`HW_DATA` 物件儲存身高/體重/年級/班級/座號/姓名/性別

### 匯出功能
- [x] **下載調整後 xlsx**：按班別分 sheet，欄位與原始上傳格式一致，800m 轉回 m.ss
- [x] **匯出體適能格式**（教育部上傳）：4–6 年級，含身高體重，資料從第 17 列開始
- [x] **匯出身高體重格式**（教育部上傳）：1–6 年級全校，體適能欄位留空
- [x] **班級名稱格式下拉選單**：可切換 `101` 或 `一年一班`，兩種匯出格式共用
- [x] **匯出前資料品質警示**：偵測學號疑似身分證、身分證空白、生日空白，列出問題學生後詢問是否繼續

### 安全性
- [x] 密碼保護頁面（sessionStorage，關閉分頁後需重新輸入）

## 3. 重要全域變數與函式

| 變數/函式 | 說明 |
|-----------|------|
| `ALL_DATA` | 所有學生體適能資料陣列 |
| `BASIC_DATA` | 學生基本資料，key = 學號 |
| `HW_DATA` | 身高體重資料，key = 學號，含 grade/cls/no/name/gender |
| `NORM` | 常模物件，key = `bend/situp/curlup/jump/run`，子 key = `男10` 等 |
| `TEST_DATE` | 測驗基準日，`new Date(2026,2,1)`，目前寫死 |
| `editedIds` | 已調整學生的 id Set（手動 + 自動） |
| `formatClsName(clsCode)` | 依下拉選單轉換班級代碼（101 / 一年一班） |
| `dataQualityReport(rows)` | 回傳資料品質問題陣列，供匯出前警示使用 |
| `adjustToPass(r)` | 將學生所有未達 25th 項目設為 NORM[key][4] |
| `getPR(score, nvals, higherBetter)` | 回傳 `{label, num}`，num 為 PR_NUMS 中的數值 |

> **getPR 邏輯說明：** 對 higher-is-better 項目，score ≥ NORM[k][4] 才能得到 PR=25（index 4 是最低及格門檻）；run 相反，score ≤ NORM.run[k][4] 才能 PR=25。

## 5. 已解決問題紀錄

| 問題 | 解決方式 |
|------|----------|
| xlsm 各班工作表格式不一致 | 用欄名偵測取代固定欄位位置 |
| 800m 時間格式 m.ss 轉換 | 轉成秒數比對常模，顯示與輸出再轉回 m.ss |
| 年級 tab 切換顯示 0 筆 | 改用 `data-grade` attribute |
| 仰臥起坐常模錯誤 | 以 fitness.org.tw 官方數據重新校正（2026-06-17） |
| 身分證誤填到學號欄 | handleBasicFile 自動偵測並對調 |
| 班級名稱不符教育部系統 | 新增班級格式下拉選單（101 / 一年一班） |
| 601/612 無出生年月日 | 無資料時依班級代碼第一碼推算年齡（6xx=12歲） |

## 6. 未來可優化的建議

### 高優先
1. **測驗日期改為可輸入**：目前寫死 `new Date(2026,2,1)`，每學年需改程式碼；應改為頁面上可設定的欄位
2. **班級統計摘要頁**：新增「班級統計」tab，顯示各班及格率一覽表，方便教師掌握全校狀況
3. **資料不齊學生清單下載**：匯出前的品質警示目前只顯示 alert，可加入「下載問題清單 xlsx」讓老師回去比對補正

### 中優先
4. **自動調整顯示說明**：調整後可在該學生列旁顯示「原始成績」tooltip，讓老師知道每個項目調整了多少
5. **列印版面**：新增 `@media print` 樣式，讓未及格學生名單可直接列印或轉 PDF
6. **多校年比較**：支援匯入上一學年資料，顯示每位學生 PR 的進退步情況

### 低優先
7. **4 年級常模缺口**：部分 4 年級學生實際年齡為 9 歲（生日較晚），目前 clamp 至 10 歲；待確認是否影響判讀
8. **班級統計摘要頁**：各班及格率、未及格人數、四項均<25th 人數一覽
9. **漸速耐力跑（PACER）支援**：目前此欄固定留空，若未來學校有施測可加入
10. **身高體重 BMI 計算**：匯入身高體重後，可順便顯示學生 BMI 與體位（教育部有相關常模）
